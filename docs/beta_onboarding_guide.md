## Using the Translation Feature

### How It Works
- Patient sends a message in Korean
- System automatically translates to English
- Staff reviews and responds in English
- System translates staff response back to Korean
- Response is sent to patient via SMS

### Best Practices
1. **Use Simple Language**: Avoid complex medical jargon in your responses
2. **Verify Critical Information**: Always double-check important details (medications, dosages)
3. **Provide Context**: When in doubt, add context to your messages
4. **Human Backup**: Always be ready to connect the patient with a human translator if needed

### Common Use Cases
- Appointment scheduling and confirmation
- Symptom description for pre-visit assessment
- Medication instructions
- Follow-up care instructions

### Enabling SMS
1. Go to Settings → Communication Channels
2. Enable SMS by toggling the switch
3. Confirm your clinic's phone number
4. Test by sending a message to our test number

### Message Templates
We provide templates for common medical communications:
- Appointment confirmation
- Appointment reminder (24 hours before)
- Follow-up request
- Prescription renewal instructions

## Troubleshooting

### Common Issues
**Q: Translation is not accurate**
A: For critical information, always confirm with the patient. Flag inaccurate translations using the feedback button.

**Q: SMS messages not sending**
A: Check your internet connection and ensure SMS service is enabled in settings.

**Q: Patient didn't receive message**
A: Verify the patient's phone number and check if messages are being blocked by their carrier.
